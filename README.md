CloudSole Angular
====================

The application demonstrates how to configure Spring MVC with AngularJS and Velocity.

Applications:

    1. Todo App
    2. Address Book
    3. Dynamic Tables
    4. Dynamic Tabs
    4. File Upload
    5. Ace Editor
    6. Restangular
    7. D3 Chart
    8. Force.com 

Run: 

    mvn tomcat:run

Open:

    http://localhost:8080/CloudSoleAngular
